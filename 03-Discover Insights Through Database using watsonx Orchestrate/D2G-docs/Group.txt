# Group

## Overview
A small lookup/dimension table defining call center agent group levels. Categorises agents into sales or technical skill levels (Level 1–3).

## Columns
| Column | Description |
|---|---|
| ID | Surrogate primary key |
| Group_ID | Business key for the group |
| Group | Group label (e.g. "Level 1 - Sales", "Level 2 - Technical") |

## Sample Data
| Group_ID | Group |
|---|---|
| 1 | Level 1 - Sales |
| 2 | Level 1 - Technical |
| 3 | Level 2 - Sales |
| 4 | Level 2 - Technical |
| 5 | Level 3 |

## Relationships
- `Group_ID` → referenced by `Call_Center_Fact.Group_ID`
